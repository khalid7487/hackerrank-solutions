## Time Complexity = o(n^3)

## Conditions: a[j] - a[i] = a[k] -a[j] = d

## Algorithms 
1. let count = 0, loop through i=0,j=i+1,k=j+1 to n
2.   a[j] - a[i] = a[k] -a[j] = d check the conditions
3. if true the count++;
